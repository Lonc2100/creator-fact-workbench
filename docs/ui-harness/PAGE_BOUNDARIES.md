# UI Page Boundaries

左侧导航决定页面边界。页面可以引用其他模块的摘要，但不能把其他模块的主工作流塞进来。

| 页面 | 只负责 | 不负责 | 主要参考 |
| --- | --- | --- | --- |
| `/calendar` 发布日历 | 周/月排期、平台版本状态、阻塞原因、发布检查 | 导入 diff、复盘写作、指标深挖 | Postiz / Mixpost |
| `/content` 内容管理 | 内容库、内容详情、平台版本编辑、转入发布日历 | 图表分析、数据导入、线索跟进 | Directus / Baserow |
| `/import` 数据导入 | 来源选择、字段识别、diff 预览、确认保存、运行记录 | 排期、内容编辑、复盘总结 | n8n executions |
| `/dashboard` 数据看板 | KPI、趋势、平台对比、内容排行、异常提醒 | 改发布状态、写复盘、导入文件 | Metabase |
| `/reviews` 周月复盘 | 保存复盘、证据 refs、行动项、下轮计划 | 复杂图表探索、导入、日历拖拽 | Evidence |
| `/leads` 线索跟进 | 线索状态、下一步、预计价值、复盘动作关联 | 内容编辑、指标分析 | 简化 CRM |
| `/ui-lab` 组件实验室 | 组件状态、视觉 token、空态/错误态/加载态 | 真实业务操作 | Storybook 思路 |

## 防混合规则

- 发布日历页可以显示“本周发布数”，但不能显示完整复盘报告。
- 数据看板可以链接到内容排行，但不能编辑内容正文。
- 周月复盘可以引用指标和日历证据，但不能承担图表探索主任务。
- 导入页必须先 preview，再保存；不能把导入结果直接塞进看板作为无审查事实。
