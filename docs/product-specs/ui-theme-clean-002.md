# UI-THEME-CLEAN-002 纯净背景主题试验

## 背景

当前米白主题可以保留，但长时间看 Dashboard 和复盘表格时可能偏糊。本任务不直接切换全站主题，只建立一个轻量 token 试验，方便后续截图对比。

## 范围

- 新增 `self-media-pure` token 主题，降低米黄色面积，提高背景纯净度。
- 在 `/ui-lab` 增加主题预览卡，对比当前米白与纯净暖白。

## 非范围

- 不切换全站默认主题。
- 不修改业务页面布局。

## 验收

- `/ui-lab` 可看到主题实验入口。
- token 仍符合 shadcn/Tailwind CSS variables 方向。
