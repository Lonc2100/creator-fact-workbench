# UI-DASHBOARD-TABLES-002 Dashboard 日周月与平台原生字段表

## 背景

当前 Dashboard 图表结构可用，但用户明确希望后续至少对齐媒体平台后台的数据表类型，并支持日数据、周数据、月数据。真实平台字段仍会持续扩展，本任务先基于内部 `metrics` 和 `metricSnapshots` 生成可验证表格。

## 范围

- 新增日 / 周 / 月指标表，字段包括内容数、曝光、点赞、评论、收藏、分享、粉丝变化、互动率。
- 新增平台原生字段预览表，按抖音、小红书、公众号、视频号、B站常见口径显示主指标、收藏类、分享/转发、粉丝/关注变化。
- 表格数据只读取内部事实源，不直接访问外部平台。

## 非范围

- 不新增真实平台 API。
- 不新增复杂筛选器。
- 不重写 Dashboard 指标计算服务。

## 验收

- `/dashboard` 仍包含 `曝光与互动趋势`。
- 新表格从内部指标聚合生成。
- `MetricDashboardGrid` 不直接 fetch，不越过 UI Harness 边界。
