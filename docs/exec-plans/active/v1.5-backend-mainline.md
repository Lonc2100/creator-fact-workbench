# Active Exec Plan: V1.5 Backend Mainline

## Scope

Implement backend facts for:

- platform versions;
- publish calendar records;
- publish records;
- dated metric snapshots;
- saved reviews;
- review action items;
- lead status progression;
- automation run records;
- evidence-backed insights.

## Ambiguities Recorded for Next Review

- Real platform CSV exports may use different headers than current presets.
- Full import diff UI needs product design before implementation.
- Calendar drag/drop first pass is implemented as a lightweight dnd-kit grid; richer month navigation, time-axis scheduling, and drag affordance polish still need product review.
- Platform-specific draft fields may need richer media attachment support later.
- Manual action item creation is not implemented yet; action items currently come from saved reviews.
- Review history has no dedicated detail route yet.

## Current Decision

Keep implementation local-first and service-owned. API/UI must only enter through Runtime.
